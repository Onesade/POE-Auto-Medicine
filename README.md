# POE-Auto-Medicine

一个用于《流放之路（Path of Exile）》的辅助工具，能够自动监测血量和魔力值变化，并在低于指定阈值时自动使用药品

## ✨ 功能简介

- 自动检测角色的血条与蓝条状态
- 根据预设阈值自动触发药品使用
- 支持最小化后台运行
- 高性能、低资源占用
